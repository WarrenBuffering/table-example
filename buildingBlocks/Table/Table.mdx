## Table

**Note. This is not a full table component implementation. `Table` is an
RDS-styled component which accepts children - to be in conjunction with
RDS-provided table component building-blocks (links provided below) or
independently.**

`Table` handles the removal of cell borders, adds spacing between rows, and
handles the scroll-y overflow for the parent `<table />` component.

`Table` can be used a standalone component, or with other RDS-provided table
building-blocks
([see full table implementation](/components/data-tables/data-table-example))
for the development of OOD styled HTML tables.
